# FineZoom

**FineZoom** is a lightweight addon for **World of Warcraft Vanilla / Turtle WoW** that provides precise and smooth camera zoom control using custom zoom steps.

Due to engine limitations in Vanilla, FineZoom works by offering configurable zoom actions instead of modifying the native mouse wheel behavior. This ensures maximum compatibility and stability while preserving the original feel of the game.
